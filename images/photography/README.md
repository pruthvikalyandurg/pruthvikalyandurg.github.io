# Photography Images

Add your photos here before pushing to GitHub Pages.

## Folder structure

```
images/photography/
  p1/   ← @pruthvikalyandurg photos (01.jpg – 06.jpg)
  p2/   ← @arrestomomentumphotography photos (01.jpg – 06.jpg)
```

## How to add photos

1. Export/save your chosen photos as JPG (recommended: 1200×1200px square, <500KB each)
2. Name them `01.jpg` through `06.jpg`
3. Drop `p1/` photos: nature, science comm, travel, plant portraits, golden hour, street
4. Drop `p2/` photos: macro florals, leaf textures, water, botanical close-ups, forest, wildlife
5. Commit and push — they appear automatically on the site

If a photo is missing, the tile shows a coloured gradient fallback instead.
